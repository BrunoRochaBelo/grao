## Fluxos de Interação

### Fluxo 1: Expandir Legenda

```
Timeline
  ↓ (tap no card)
Legenda Expandida
  ├─ [Editar] → Edição de Momento
  ├─ [Compartilhar] → Toast "Em breve"
  ├─ [Excluir] → Modal Confirmação → Deletado
  └─ [Fechar] → Volta para Timeline
```

### Fluxo 2: Abrir Fullscreen

```
Timeline
  ↓ (duplo-tap em imagem)
Fullscreen Viewer
  ├─ Swipe Lateral → Próxima/Anterior Imagem
  ├─ Pinch-to-Zoom → Zoom até 3x
  ├─ Duplo-tap → Reset Zoom
  ├─ Swipe Down → Fecha (volta Timeline)
  ├─ [Editar] → Edição
  ├─ [Compartilhar] → Modal Compartilhamento
  └─ [Excluir] → Confirmação → Deletado
```

### Fluxo 3: Filtrar

```
Timeline
  ↓ (clique em chip de capítulo)
Timeline Filtrada
  ├─ Mostra apenas momentos do capítulo
  ├─ Se há placeholders → mostra também
  ├─ Botão "✕ Limpar" aparece
  └─ Clique "✕ Limpar" → Volta ao completo
```

## Animações

| Evento                  | Duração | Easing      | Feedback           |
| ----------------------- | ------- | ----------- | ------------------ |
| Expande legenda         | 200ms   | ease-out    | Slide down         |
| Fecha fullscreen        | 300ms   | ease-in     | Fade out           |
| Menu contextual         | 150ms   | spring      | Scale 0.9→1        |
| Troca imagem fullscreen | 200ms   | ease-in-out | Fade + haptic 10ms |
| Zoom completo           | 400ms   | spring      | Inércia suave      |
| Fade entre meses        | 300ms   | ease-in-out | Suave              |
| Limpar filtros          | 300ms   | ease-out    | Fade in "botão"    |

## Responsividade

### Mobile (< 640px)

- Cards full-width
- Chips empilham se necessário
- Filtros scroll horizontal
- Texto ajustado (14-16px)

### Tablet (640px - 1024px)

- Cards com padding 16px
- Chips em 2 linhas se necessário
- Layout otimizado

### Desktop (> 1024px)

- Max-width 896px (centrado)
- Cards com espaçamento generoso
- Filtros lado a lado com espaço

---

## Tonalidade & Emoção

- **Cada card é um "tesouro"** → Hover effect suave
- **Cada gesto é intencional** → Animações lentas e significativas
- **Cada exclusão é cuidadosa** → Modal de confirmação com tom ternurento
- **Cada sucesso é celebrado** → Toasts com emojis ("🌸" "🧸" "💭" "⏳")
- **Cada momento persiste** → Transições suaves preservam sensação de continuidade
